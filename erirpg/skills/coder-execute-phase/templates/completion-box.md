# Completion Box Template

Use this template when verification passes.

```
╔════════════════════════════════════════════════════════════════╗
║  ✓ PHASE {N} EXECUTED AND VERIFIED                             ║
╠════════════════════════════════════════════════════════════════╣
║  Plans completed: {X}/{Y}                                      ║
║  Verification: passed                                          ║
║  Commits: {commit hashes}                                      ║
╚════════════════════════════════════════════════════════════════╝

## ▶ NEXT: User Acceptance Testing

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Type:  /clear
2. Then:  /coder:init
3. Then:  /coder:verify-work {N}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**Replace:**
- `{N}` — Phase number
- `{X}/{Y}` — Plans completed / total plans
- `{commit hashes}` — Short hashes from this execution
