# EriRPG Help Topics

## Commands
| Topic | Description |
|-------|-------------|
| `quick` | Quick fix mode for single files |
| `execute` | Full agent runs with verification |
| `discuss` | Goal clarification discussions |
| `learn` | Store knowledge about modules |
| `recall` | Retrieve stored knowledge |
| `decide` | Log decisions |
| `defer` | Capture ideas for later |
| `spec` | Manage specifications |
| `plan` | Manage execution plans |
| `verify` | Run verification tests |
| `roadmap` | Long-term planning |
| `impact` | Impact analysis |
| `cleanup` | Clean up stale state |
| `reset` | Reset to clean state |
