---
name: eri:learn
description: Store knowledge about a module after reading it
argument-hint: "<project> <file>"
---

<action>
Run this command to store knowledge:

```bash
python3 -m erirpg.cli learn $ARGUMENTS
```

To retrieve later:
```bash
python3 -m erirpg.cli recall <project> <file>
```
</action>
